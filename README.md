# E-Station Clone Project

Этот проект является визуальным клоном (концептом) сайта компании **E-Station**, специализирующейся на решениях для зарядки электромобилей.

## 🔗 Оригинал
Первоисточник дизайна и идеи: [https://e-station.tech/](https://e-station.tech/)

## 🛠 Стек технологий
- **HTML5**: Структура контента.
- **CSS3**: Стилизация, Flexbox, CSS Grid и адаптивная верстка.
- **Google Fonts**: Современные шрифты для лучшей читаемости.

## 📋 Описание
Проект воссоздает ключевые элементы интерфейса оригинального сайта:
* Темная минималистичная цветовая схема.
* Акцентный зеленый цвет (`#2ecc71`), подчеркивающий экологичность.
* Адаптивная сетка карточек с услугами.
* Полноэкранный Hero-блок с призывом к действию.

## 🚀 Как запустить
1. Склонируйте репозиторий или скачайте файл `index.html`.
2. Откройте `index.html` в любом современном браузере (Chrome, Firefox, Safari).
